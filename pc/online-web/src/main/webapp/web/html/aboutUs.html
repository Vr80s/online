关于我们

说在前面：

中国已成为全球第二个经济体，中华民族伟大复兴需要以中华文化发展繁荣为傲娇。 中医学是中国传统文化不可分割的中药组成部分，是最能体现中华文化优秀特质的部分。 在中华民族伟大复兴的历史机遇面前，中医的春天来了。

熊猫中医云课堂：

熊猫中医云课堂，是中国第一家以直播形式服务于中医领域的传承和交流平台。 平台由资深互联网专家、中医传承经营、中医大数据人才和优秀的中医师组成。为了更好 地服务人类健康而致力于发扬中医药文化、培育中医药人才、助力中医药产业。

熊猫中医共享基地：

线下还建立有熊猫中医传承基地，基地位于全国空气质量排名第一的海口市演丰互联网小 镇，基地以共享经济的模式打造中医传承基地，专业的教室、完善的配套、走心的服务。

线上+线下，中医交流全方位：

平台会定期邀请知名中医大咖定期来基地授课交流，线上也会通过平台向全球中医爱好者 进行互动直播。使中医文化的传承和交流全方位，再也不受时间、空间的局限。

有线下课程吗？

您对中医学习有不同的需求和习惯。正因如此，我们持续开设线下课程 ，详情欢迎咨询客服人员。

有哪些知识阶层的中医课程？

我们对所有课程根据学识储备进行个性化的划分，保证每个认知阶段的用户都能找到心仪的课程。

没有中医基础，可以学习课程吗？

零基础也能学中医！许多学员推荐“古籍经典”系列以及【小宝中医带你快速入门学针灸】课程作为他们的入门必备。

学习中需要大家准备什么工具？

所有熊猫中医课程都有建议配套使用的教具。例如【小宝中医带你快速入门学针灸】系列课程，老师推荐需要配合我们的【新手包】来进行学习。

【新手包】中有什么内容？

【新手包】中包含有针具、练针包、课程推荐练针板、用于针灸经络记忆的《外治基础手册》，以及讲师所著《医路十年》。

请联系我们平台的客服，获取购买【新手包】的详细信息。
